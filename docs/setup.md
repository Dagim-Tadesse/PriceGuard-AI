\# Setup

This repo does **not** force a specific framework. Each team member can use tools they are comfortable with.

\## Shared requirement

- The backend must implement the endpoints and JSON shapes defined in \`docs/api.md\`.
- The AI/prediction module must produce the stable output contract described in \`docs/api.md\`.

\## Local dev (recommended)

1. Start your backend API server.
2. Confirm the backend base URL (example: \`http://127.0.0.1:8000/api\`).
3. Configure the frontend to call that base URL.
4. Verify the demo flow in \`docs/demo.md\`.
